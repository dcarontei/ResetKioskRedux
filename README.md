# ResetKioskRedux
Firefox addon that resets the current tab to the browser home page after a configurable inactivity. For use with Firefox's Kiosk mode.

Inspired by https://github.com/cuetus/resetkiosk. Re-written as a webextension manifest v3 addon.

## Downloading
Get it here: https://addons.mozilla.org/en-US/firefox/addon/resetkioskredux/
